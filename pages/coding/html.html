Hypertext Markup Language

RandomCollection is hosted via GitHub Pages. The corresponding GithHub repository can be found here. In general, modern web design relies on three building blocks:

  • Structure: Hypertext Markup Language (HTML)
  • Style: Cascading Style Sheets (CSS)
  • Interactivity: JavaScript
RandomCollection does not incorporate any JavaScript and is static in nature. The usage of responsive web design is very limited and a correct appearance of the website across different devices cannot be guaranteed.
