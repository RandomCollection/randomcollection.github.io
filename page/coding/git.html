Git

Git is a software. In particular, Git is an open source, distributed version control system. In contrast, GitHub is a service. In particular, GitHub is a centralised source code hosting platform for collaborating on Git repositories.

Commands

  • git add <file>: add <file> to the staging area
  • git add .: add all files to the staging area
  • git add -all: add all changes to the staging area including, for example, the deletion of files
  • git branch: show local branches
  • git branch new <branch>: create a new branch <branch>
  • git branch -a: show all local branches
  • git branch -D <branch>: delete branch <branch>
  • git branch -m <name_old> <name_new>: change the name of branch <name_old> from <name_old> to <name_new>
  • git branch -r: show remote branches
  • git checkout <branch>: switch to branch <branch>
  • git checkout <file>: undo changes to the unstaged file <file>, i.e. pull the file <file> back to the staging area
  • git checkout .: undo changes to all unstaged files
  • git checkout -b <branch>: create a new branch <branch>
  • git clone <name_of_GitHub_repository>.git: clone repository to the local machine
  • git commit -m "<message>": commit changes with message "<message>"
  • git config --global user.email 'johndoe@google.com': set user mail
  • git config --global user.name 'John Doe': set user name
  • git init: initialise Git (creates a hidden ".git" file)
  • git log: show the log (enter "q" to quit)
  • git merge <branch>: merge with branch <branch>
  • git merge <source> <destination>: merge from <source> to <destination>
  • git mv <name_file> <name_folder>: move file <name_file> to <name_folder>
  • git mv <name_old> <name_new>: rename folder <name_old> from <name_old> to <name_new>
  • git rm -r <folder>: delete folder <folder>
  • git stash: temporarily save files, for example, to move from one branch to another
  • git status: show the status
  • git --version: show the Git version

Other

A .gitignore file specifies intentionally untracked files that Git should ignore.
